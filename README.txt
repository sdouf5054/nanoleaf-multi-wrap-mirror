Nanoleaf Multi-Wrap Mirror
==========================

Nanoleaf PC Screen Mirror Lightstrip을 자르지 않고 모니터 뒷면에 여러 겹 감아 쓸 수 있도록 만든 앱입니다.

N7601K00-1LS4D 모델 바탕으로 테스트, 제작하였습니다. (카메라 사용하는 제품과 호환 안됨, NF082K07-40LS, NF082K07-52LS 등)

호환 제품 링크 : https://int-shop.nanoleaf.me/products/pc-screen-mirror-lightstrip


요구사항
--------
- Windows 10/11 (64비트)
- Nanoleaf 4D LED 스트립 (USB HID 연결)


실행
----
압축을 해제하고 NanoleafMirror.exe를 실행하세요.
정상 작동을 위해 .exe 파일 포함 폴더 내부 파일을 임의로 옮기지 마세요. 폴더 자체는 옮겨도 됩니다.
다른 곳에서 열고 싶으시면 .exe 바로가기를 만들어 활용하세요.
config.json과 presets/ 폴더가 exe와 같은 위치에 자동 생성됩니다.

※ Windows SmartScreen 경고가 뜨면 "자세히 → 실행"을 클릭하세요.
   코드 서명이 없는 개인 빌드이기 때문에 발생하는 정상 경고입니다.


처음 사용할 때 — LED 레이아웃 설정 (필수)
-----------------------------------------
이 앱은 LED 스트립을 모니터 둘레에 감아서 사용하는 구조입니다.
모니터 크기와 감는 방식에 따라 LED 배치가 다르므로,
처음 한 번 코너 위치를 설정해야 합니다.

설정 시 번호의 경우, 
LED 0번이 첫 번째, USB 연결부와 가장 가까운 부분이고, 
74번이 마지막, USB 연결부와 가장 가까운 부분, 잘라도 되는 끝 부분입니다.


1. "LED 설정" 탭을 엽니다.

2. 기본 설정:
   - LED 수: 스트립의 전체 LED 개수 (기본 75개, 잘랐을 경우 개수 맞춰 설정 가능)
   - 바퀴 수: 모니터 둘레를 몇 바퀴 감았는지 (24인치 기준 상좌우 2바퀴, 하단 1바퀴)
   - 시작 위치: LED 스트립이 시작되는 모니터 코너 (기본 좌하, 제가 이렇게 써서 그렇고 본인 세팅에 맞게 바꾸셔도 됩니다)

3. "LED 연결" 버튼을 누른 뒤, "자동 스캔" 또는 "수동 모드"를 시작합니다.
   LED가 0번(USB 연결부와 가장 가까운 부분)부터 하나씩 순서대로 점등됩니다.

4. 모니터 코너에 해당하는 LED가 켜질 때마다
   "이 LED를 코너로 기록" 버튼을 누릅니다.

   예시 (24인치 모니터 기준, 1+3/4 바퀴, 좌하 시작):
   ┌─────────────────────────────────┐
   │  바퀴 1                  									│
   │  시작(좌하) 74 → 좌상 66 → 우상 52 → 우하 44 → 끝점(좌하) 29		│
   │  바퀴 2 (끝점이 다음 바퀴의 시작)              					│
   │  끝점(좌하) 29 → 좌상 21 → 우상 8 → 우하 0 → 끝점 0			│
   └─────────────────────────────────┘

   - 바퀴 1: 시작 코너(좌하)부터 시계방향으로 각 코너의 LED 번호를 기록
   - 바퀴 1의 끝점 = 바퀴 2가 시작되는 위치
   - 바퀴 2도 같은 방식으로 코너를 기록
   - 마지막 바퀴의 끝점 = 스트립의 마지막 LED 번호

5. 모든 코너를 기록한 뒤 "세그먼트 생성" → "LED 레이아웃 저장"을 누릅니다.

6. "컨트롤" 탭으로 돌아가서 "시작" 버튼을 누르면 미러링이 시작됩니다.


탭 설명
-------
- 컨트롤: 미러링/오디오/미디어 토글, 밝기, 프리셋, LED 프리뷰
- 색상 보정: 화이트밸런스, 감마, 채널 믹싱 조정
- LED 설정: 코너 캘리브레이션, 세그먼트 생성
- 옵션: 핫키, 트레이, 시작프로그램, 테마 설정


오디오 모드
-----------
- Pulse: 베이스에 맞춰 전체 밝기 반응
- Spectrum: 16밴드 주파수를 LED 둘레에 매핑
- Bass Detail: 저역(20~500Hz)을 16밴드로 세밀하게 표현
- Wave: 베이스 온셋마다 펄스가 아래→위로 이동
- Dynamic: 비트마다 LED 둘레에서 파원(리플) 발생
- Flowing: 화면 색상을 팔레트로 추출해 LED 둘레를 순환


문제 해결
---------
- LED가 연결 안 됨: USB 케이블 확인, 장치 관리자에서 Vendor/Product ID 확인
- 화면 캡처 안 됨: 다른 캡처 프로그램(OBS 등)이 Desktop Duplication을 점유 중인지 확인
- 오디오 반응 없음: 옵션 탭에서 오디오 디바이스가 올바른지 확인
- 첫 실행 시 느림: 정상입니다. 두 번째 실행부터는 빨라집니다.


설정 파일
---------
- config.json: 모든 설정이 자동 저장됩니다. 직접 편집도 가능합니다.
- presets/: 프리셋 파일이 저장됩니다. 다른 PC로 복사해서 사용할 수 있습니다.
